
# basicsof3D

A small pygame program used to demostrate projection of 3D shapes on a 2D plane.

![Cipherme](https://github.com/abelureste/basicsof3d/assets/3d_demo.gif)

Credit to [Tsoding's video on demystifying 3D graphics](https://www.youtube.com/watch?v=qjWkNZ0SXfo&list=LL&index=5). The code in this project was adapted from his video.

## Features

Three shapes are demonstrated:

- Cube
- Polyhedron
- Icosahedron

## How to Run

Download the code and run the 3d.py file using your choice of python environment.
